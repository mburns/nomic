# Rule

A rule-change is adopted if and only if the vote is at least two-thirds in the affirmative (`+1`) among eligible voters.

```javascript
let v = number of votes
let a = number of affirmative votes

let p = a / v

if p >= (2/3), proposal is adopted
```
